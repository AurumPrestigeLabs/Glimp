# 🗺️ Glimp - Project Roadmap

This document outlines the planned features and future direction for **Glimp**. We want to evolve from a simple desktop utility into a versatile productivity ecosystem.

## 🟢 Phase 1: Foundation (Current - v2.1)
* [x] Core Note-taking engine (AutoIt).
* [x] Screenshot acquisition and linking.
* [x] Tagging and color-coding system.
* [x] Multi-format export (HTML, CSV, TXT).
* [x] Portable version (No installation).

## 🟡 Phase 2: Enhanced Desktop Experience (Short-term)
* [ ] **Search & Filter:** Advanced search bar to find notes by content or date.
* [ ] **Rich Text Support:** Basic formatting (Bold, Italic) inside notes.
* [ ] **Auto-Updater:** Check for new versions directly from the app.
* [ ] **Custom Themes:** Dark mode and UI customization.

## 🟠 Phase 3: Connectivity & Cloud (Mid-term)
* [ ] **Cloud Sync:** Option to sync notes via Google Drive or Dropbox API.
* [ ] **Encryption:** End-to-end encryption for sensitive notes stored in the cloud.
* [ ] **Web Clipper:** Browser extension to send snippets directly to Glimp.

## 🔴 Phase 4: Mobile Ecosystem (Long-term)
* [ ] **Glimp Mobile App:** Companion app (Android/iOS) to read and take notes on the go.
* [ ] **Push Notifications:** Reminders for time-sensitive notes.
* [ ] **Cross-platform Sync:** Seamless transition between Desktop and Mobile.
